## Staging

```bash
# Show modified files in working directory, staged for the next commit
git status

# Stage a file to next commit
git add [file]

# Unstage a file from the next commit
git reset [file]

# Unstage all files from the next commit
git reset --hard

# Show diff of what is changed but not staged
git diff

# Show diff of what is staged for the next commit
git diff --staged
git diff --cached

# Commit staged changes to repository
git commit -m "[commit message]"
```
