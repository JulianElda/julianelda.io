## Fetch and pull

```bash
# Fetch changes from remote repository
git fetch

# Fetch changes and remove deleted branches
git fetch --prune

# Set git to always prune on fetch
git config --global fetch.prune true

# Fetch and merge changes from remote repository
git pull

# Fetch and rebase changes from remote repository
git pull --rebase

# Fetch and fast-forward only
git pull --ff-only
```
